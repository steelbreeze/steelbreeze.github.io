Choice

Choice:
  • Defined in state.ts:70

Turns the pseudo state into a dynamic conditional branch: the guard conditions of the outgoing transitions will be evaluated after the transition into the pseudo state is traversed.

DeepHistory

DeepHistory:
  • Defined in state.ts:73

Turns on deep history semantics for the parent region: second and subsiquent entry of the parent region will use the last known state from the active state configuration contained withn the state machine instance as the initial state; this behavior will cascade through all child regions.

Initial

Initial:
  • Defined in state.ts:76

Turns the pseudo state into an initial vertex, meaning is is the default point when the parent region is entered.

Junction

Junction:
  • Defined in state.ts:79

Turns the pseudo state into a static conditional branch: the guard conditions of the outgoing transitions will be evaluated before the transition into the pseudo state is traversed.

ShallowHistory

ShallowHistory:
  • Defined in state.ts:82

Turns on shallow history semantics for the parent region: second and subsiquent entry of the parent region will use the last known state from the active state configuration contained withn the state machine instance as the initial state; this behavior will only apply to the parent region.

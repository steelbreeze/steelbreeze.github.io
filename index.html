Open source software for transaction processing

This site is dedicated to promoting a collection of open source software tools specialising in reliable and efficent transaction processing. The tools today comprise of:

  • delegate: an immutable multicast delegate for TypeScript and JavaScript;
  • graph: an emerging set of graph functions for TypeScript and JavaScript;
  • state: a finite state machine for TypeScript, JavaScript and C#.

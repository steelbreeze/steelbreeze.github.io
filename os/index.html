steelbreeze is a strong advocate of open source software. We actively develop and publish software in the field of executable finite state machines and a few libraries in support of them.

Current projects

  • delegate: an immutable multicast delegate for TypeScript and JavaScript
  • graph: an emerging set of graph functions for TypeScript and JavaScript;
  • state: an executable finite state machine for TypeScript, JavaScript and C#.
